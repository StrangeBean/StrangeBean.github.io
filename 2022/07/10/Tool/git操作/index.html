Git命令

July 2022

常见命令

  1. 克隆代码： git clone https://gitee.com/strangebean/dou.git

  2. 初始化git仓库 ： git init

  3. 查看本地git管理状态：git status

  4. 添加文件到git管理：git add filename

    添加所有文件： git add .

  5. 提交文件：git commit -m “add new file”

  6. 查看提交信息： git log

  7. 查看当前分支： git branch

  8. 新建分支并切换到新分支： git checkout -b <新分支名>

  9. 切换分支：
    8.1 git switch <目标分支名>
    8.2 git checkout <目标分支名>

    git switch 为高版本支持的命令，低版本可能不支持

  10. 添加远程库：git remote add origin https://gitee.com/strangebean/dou.git

    origin: 远程仓库别名

  11. 查看远程库列表：git remote -v

  12. 修改远程库别名 git remote rename oldName newName

  13. 获取远程仓库代码到本地（不合并）： git fetch origin

  14. 合并远程dev分支到本地分支：git rebase origin/dev

  15. 推送到远程： git push -u origin master:dev

    git push <远程主机名> <本地分支名>:<远程分支名>
    -u：添加此参数后，以后推送使用git push即可

  16. 贮存本地修改：git stash

    贮存后本地就恢复到上一个提交状态，分支干净无任何修改

  17. 恢复本地修改：git stash pop

标准开发git提交流程

  1. 贮存本地修改：git stash
  2. 获取远程仓库代码到本地（不合并）： git fetch origin
  3. 合并远程dev分支到本地分支：git rebase origin/dev
  4. 恢复本地修改：git stash pop
    4.1 若发生冲突，选择更改，然后 git add .
  5. 提交代码：git commit -m “update file”
  6. git push origin master:dev
